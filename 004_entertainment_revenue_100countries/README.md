# 🎬 K-POP & Global Entertainment Companies Revenue Dashboard

연습용으로 첫 시작을 다지겠습니다.

본 프로젝트는 한국을 포함한 글로벌 엔터테인먼트 기업 100개를 대상으로, 기업 유형·지역·주요 아티스트·매출 데이터를 집계하고 시각화한 Power BI 대시보드입니다.  
K-POP 메이저부터 콘텐츠 제작사, OTT, 공연 기획, 배우 매니지먼트까지 다양한 기업을 포함하며, 글로벌 시장 분포를 한눈에 파악할 수 있도록 구성했습니다.

---

## 📌 프로젝트 개요

- **데이터 출처**: 직접 정리한 100개 엔터테인먼트 기업 리스트 (한국, 일본, 미국, 중국 등)
- **분석 대상**:
  - 기업명
  - 지역 (국가)
  - 기업 유형 (K-POP, OTT, 콘텐츠 제작 등)
  - 주요 아티스트/사업
  - 연습용 매출 수치 (랜덤 생성)
- **목적**: Power BI 연습 및 이커머스 BI 사고방식 훈련

---

## 📊 대시보드 구성

| 시각화 | 설명 |
|--------|------|
| KPI 카드 | 전체 기업 수 / 총 매출 / 평균 매출 / Top 10 기업 매출 비중 |
| 막대 차트 | 기업별 매출 TOP 10 |
| 누적 막대 차트 | 기업 유형 + 지역별 매출 분포 |
| 도넛 차트 | 지역별 매출 비중 |
| 슬라이서 | 기업 유형 / 지역 / 주요 아티스트 필터링 가능 |
