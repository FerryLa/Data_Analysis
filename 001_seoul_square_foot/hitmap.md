## 📊 1. 10년간 변화한 특징

**📌 주의:** 아래 수치는 실제 시장 패턴을 반영한 시뮬레이션입니다.

### **숫자로 보는 극적 변화**

| 연도 | 평균 평당가 | 상승률 |
|------|------------|--------|
| 2015년 | 5.2억원 | - |
| 2020년 | 6.8억원 | +30.8% |
| 2025년 | 9.4억원 | +80.8% |

**핵심:** 10년간 평균 **80% 상승**, 연평균 **8.1%**

---

### **특징 1: 강남3구 독주 → 격차 확대**

**2015년:**
- 서초구: 6.1억 vs 강북구: 2.0억
- 격차: **3.0배**

**2025년:**
- 서초구: 12.2억 vs 강북구: 3.6억
- 격차: **3.4배**

→ **부익부 빈익빈**: 고가 지역이 더 빠르게 상승 (+100% vs +80%)

---

### **특징 2: 한강 프리미엄 강화**

히트맵에서 명확히 보이는 **한강변 빨간색 띠**:

| 지역 | 2015년 | 2025년 | 상승률 |
|------|--------|--------|--------|
| 용산 | 5.1억 | 10.3억 | +102% |
| 마포 | 4.3억 | 8.9억 | +107% |
| 성동 | 3.8억 | 7.6억 | +100% |

→ 한강 인접 지역은 **평균보다 20%p 더 상승**

---

### **특징 3: 지하철역 500m 이내 황금 입지**

역세권 vs 비역세권 격차:

```
지하철 500m 이내:  평균 9.8억원 (+95%)
지하철 1km 이상:   평균 6.2억원 (+65%)

차이: 3.6억원 (약 1.6배)
```

→ **역세권 = 최고의 투자 안정성**

---

### **특징 4: 강북 vs 강남, 절대 격차는 커짐**

**2015년:**
- 강남3구 평균: 5.3억
- 강북5구 평균: 2.3억
- 차이: **3.0억**

**2025년:**
- 강남3구 평균: 10.6억
- 강북5구 평균: 3.9억
- 차이: **6.7억**

→ 상승률은 비슷해도 **절대 금액 차이는 2배 이상 벌어짐**

---

### **특징 5: 2020년 이후 가속화**

```
2015→2020년 (5년): +30.8% (연평균 5.5%)
2020→2025년 (5년): +38.5% (연평균 6.7%)
```

→ 최근 5년이 더 빠르게 상승 **(저금리 + 유동성 증가)**

## 🏠 2. 아파트 가격 결정 핵심 4가지 (간략)

**📌 주의:** 아래 수치는 일반적인 시장 연구를 기반으로 한 추정값입니다.

### **(1) 지하철 접근성 - 82.5%** 🥇

**한 줄 요약:** 지하철역 1km 가까워질 때마다 **+8,000만원**

**왜 중요한가:**
- 서울 생활 = 대중교통
- 출퇴근 시간 = 삶의 질
- 역세권 = 상권 발달

**투자 가이드:**
```
✅ 추천: 지하철 500m 이내
⚠️ 주의: 지하철 1km 이상
```

---

### **(2) 평수 - 8.2%** 🥈

**한 줄 요약:** 33㎡(10평)당 **+1.2억원** 차이

**왜 중요한가:**
- 가족 구성원 필수 요건
- 중대형 = 재건축 가치
- 대단지 = 평수 다양성

**투자 가이드:**
```
실거주: 중형 59~84㎡ (24~34평)
투자용: 중대형 85㎡+ (34평+)
```

---

### **(3) 건물 연령 - 6.8%** 🥉

**한 줄 요약:** 준공 1년마다 **-1.5%**, 단 15년 이상은 역전

**왜 중요한가:**
- 신축 = 최신 설비
- 노후 = 재건축 기대
- U자 곡선 (신축↑ 중간↓ 노후↑)

**투자 가이드:**
```
✅ 추천: 5년 이내 신축 OR 20년 이상 노후
⚠️ 주의: 10~15년 중간 연령대
```

---

### **(4) 한강 접근성 - 2.5%** 🏅

**한 줄 요약:** 한강변 500m 이내 **+5~10%** 프리미엄

**왜 중요한가:**
- 조망권 + 쾌적성
- 산책로 + 공원
- 희소성 (한강은 하나)

**투자 가이드:**
```
✅ 추천: 한강변 고층 (조망 확보)
⚠️ 주의: 한강변 저층 (조망 제한)
```

---

### **📌 우선순위 정리**

```
1순위: 지하철역 500m 이내      (필수)
2순위: 중형 이상 평수           (중요)
3순위: 5년 이내 신축 OR 노후    (선택)
4순위: 한강 인접                (플러스)
```

**황금 조합:** 역세권 + 중형 + 신축 + 한강변 = **최고가 입지**

## 🎨 3. 히트맵의 장점과 활용방법

### **히트맵이란?**

데이터를 **색깔**로 표현하는 시각화 기법
- 🟢 초록 = 저렴
- 🟡 노랑 = 보통
- 🔴 빨강 = 비쌈
- 🟣 보라 = 초고가

---

### **장점 1: 3초 만에 이해** ⚡

**표 (전통적 방식):**
```
강남: 10.4억, 서초: 12.2억, 송파: 8.5억, 
강동: 7.0억, 양천: 7.8억...

→ 25개 구 읽고 비교하는데 3분 소요
```

**히트맵:**
```
빨간색 = 비쌈
초록색 = 저렴

→ 3초 만에 파악 완료!
```

**효율성 60배 향상 (3분 → 3초)**

---

### **장점 2: 공간 패턴 발견** 🗺️

**표로는 못 보는 것:**
```
❌ 강남3구가 붙어있다는 사실
❌ 한강 따라 가격대 형성되는 패턴
❌ 북동쪽으로 갈수록 저렴해지는 경향
```

**히트맵으로 즉시 발견:**
```
✅ 강남 중심 빨간 클러스터
✅ 한강변 주황~빨강 띠
✅ 강북으로 갈수록 초록색 그라디언트
```

---

### **장점 3: 시간 변화 추적** ⏰

**10년 색상 변화:**
```
2015년: 전체적으로 초록~노랑 (저렴)
   ↓
2020년: 강남 중심 주황~빨강 (상승)
   ↓
2025년: 전체가 노랑~빨강 (폭등)
```

**색깔 진해지는 속도 = 가격 상승 속도**

---

### **장점 4: 정보 압축** 📊

**원본 데이터:**
```
25개구 × 3개연도 × 5개지표 = 375개 숫자
```

**히트맵:**
```
1장의 이미지
```

**정보 압축률: 99.7%**
