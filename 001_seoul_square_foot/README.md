### **1. 데이터 생성 방식**

**실제 시장 연구 기반 시뮬레이션:**

```python
# 서울 25개구 기준 가격 (2020년 기준)
districts_data = {
    '강남': {'price_2020': 95000, 'premium': 1.10},
    '서초': {'price_2020': 105000, 'premium': 1.12},
    '송파': {'price_2020': 78000, 'premium': 1.05},
    ...
}

# 연도별 상승률 (시장 추세 반영)
growth_rates = {
    2015: 1.00,   # 기준년도

    2025: 1.80    # +80% 상승
}
```

**근거:**

- 한강, 지하철 효과 연구 논문 참고

### **2. 샘플 7,500건의 의미**

**"샘플 7,500건" = 히트맵 시각화를 위한 좌표점**

```
25개 구 × 100개 포인트 × 3개 연도 = 7,500개 데이터 포인트
```

**이것은:**
- ✅ 육각형 히트맵을 그리기 위한 가상 좌표
- ✅ 각 구 중심에서 랜덤하게 분포 생성
- ✅ 부드러운 색상 그라디언트 표현 목적

**이것이 아님:**
- ❌ 실제 아파트 거래 건수
- ❌ 실거래가 샘플 개수
- ❌ 통계적 대표성을 가진 표본

**비유:**
```
실제 아파트 = 밤하늘의 별
7,500개 샘플 = 별자리를 그리기 위해 찍은 점
```

   - 연도별 성장률 추정
